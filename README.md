# Tic-Tac-Toe AI

A tkinter Tic-Tac-Toe game written in Python.

## Features

- Player vs Player
- Easy AI (Random)
- Medium AI (50% chance of random or Minimax move)
- Hard AI (40% chance of random, and 60% chance of Minimax move)
- Impossible AI (100% Minimax moves)
- GUI 

## Concepts Used

- Recursion
- Backtracking
- Minimax Algorithm
- Game Trees

## How to Run

```bash
python gui.py
```

## What I Learned

- Recursive problem solving
- State-space search
- Backtracking
- Designing game logic separately from UI
